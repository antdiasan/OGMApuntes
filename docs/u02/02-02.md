# Vídeos y audios en HTML

Los elementos multimedia permiten integrar contenido audiovisual directamente en una página web.

---

## 1. Insertar un vídeo

La etiqueta `<video>` se utiliza para reproducir archivos de vídeo.

```html
<video width="480" controls>
  <source src="video.mp4" type="video/mp4">
  <source src="video.webm" type="video/webm">
  Tu navegador no soporta la etiqueta <code>video</code>.
</video>
```

### Atributos más usados

- `controls`: muestra los botones de reproducción.
- `autoplay`: inicia el vídeo automáticamente.
- `loop`: repite el vídeo.
- `muted`: lo inicia sin sonido.
- `poster`: muestra una imagen previa antes de reproducirlo.

---

## 2. Insertar un audio

La etiqueta `<audio>` reproduce sonidos o música.

```html
<audio controls>
  <source src="audio.mp3" type="audio/mpeg">
  <source src="audio.ogg" type="audio/ogg">
  Tu navegador no soporta la etiqueta <code>audio</code>.
</audio>
```

---

## Subtítulos y accesibilidad

Para mejorar la accesibilidad, se pueden incluir **subtítulos** en los vídeos:

```html
<video controls>
  <source src="video.mp4" type="video/mp4">
  <track src="subtitulos.vtt" kind="subtitles" srclang="es" label="Español">
</video>
```

- Usa el atributo `kind="subtitles"` para subtítulos traducidos o `kind="captions"` para texto descriptivo.
- Asegúrate de que los archivos `.vtt` tengan buena sincronización.
- Proporciona siempre una transcripción textual si el contenido del vídeo o audio es esencial.

---

## 3. Buenas prácticas

- Proporcionar siempre **más de un formato** (`.mp4`, `.webm`, `.ogg`) para asegurar compatibilidad entre navegadores.
- Añadir **texto alternativo o descripción** cuando el contenido audiovisual sea fundamental.
- Evitar la **reproducción automática con sonido**.
- Utilizar subtítulos o transcripciones para mejorar la **accesibilidad**.
