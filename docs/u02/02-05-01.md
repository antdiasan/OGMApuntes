# 1. Validación de HTML

Antes de publicar una página web es fundamental **validar el código HTML** para asegurarnos de que cumple con los estándares del W3C y que funcionará correctamente en la mayoría de los navegadores.

---

## El validador del W3C

El **W3C Markup Validation Service** es la herramienta oficial para comprobar si un documento HTML está bien formado y cumple con las normas.

- Sitio web: [https://validator.w3.org](https://validator.w3.org)  
- Permite validar:
  - Introduciendo la **URL** de una página publicada.  
  - Subiendo un **archivo local**.  
  - Pegando el **código fuente** directamente.  

**Ejemplo de uso:**
1. Accede a [validator.w3.org](https://validator.w3.org).  
2. Escribe la URL de tu página o sube el archivo HTML.  
3. Haz clic en **Check**.  
4. El sistema devolverá los errores y advertencias encontrados.  

---

## Errores comunes detectados

- Etiquetas no cerradas (`<p>`, `<li>`, `<table>`).  
- Atributos mal escritos (`bgcolour` en lugar de `bgcolor`).  
- Jerarquía incorrecta de encabezados (`<h1>` seguido de `<h4>`).  
- Uso de atributos obsoletos (`align`, `border`, `valign`).  
- Caracteres especiales sin entidad (`<`, `>`, `&`).  
- Falta de `alt` en imágenes.  

**Ejemplo de error común:**
```html
<p>Esto es un párrafo
<p>Otro párrafo sin cerrar correctamente</p>
```

---

## Importancia de validar

- Garantiza que el documento es **compatible con los navegadores** actuales.  
- Evita problemas de accesibilidad.  
- Facilita el mantenimiento y comprensión del código.  
- Ayuda al SEO (los buscadores prefieren páginas bien estructuradas).  

---

## Buenas prácticas en validación

| Buena práctica |
|----------------|
| <span style="color:darkred">**Validar siempre el código antes de publicarlo.**</span> |
| <span style="color:darkred">**Corregir todos los errores, aunque el navegador muestre la página.**</span> |
| <span style="color:darkred">**Revisar también las advertencias (warnings).**</span> |
| <span style="color:darkred">**Evitar atributos obsoletos y usar HTML5 válido.**</span> |
| <span style="color:darkred">**Comprobar caracteres especiales y uso de `alt` en imágenes.**</span> |
