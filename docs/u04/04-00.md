# Introducción a la maquetación web

En esta unidad vamos a comenzar a trabajar con la **maquetación web**, es decir, con la forma en la que se **organizan y colocan los elementos de una página web** utilizando CSS.

Hasta ahora hemos aprendido a aplicar estilos (colores, tamaños, bordes, fuentes…), pero en este punto damos un paso más:  
aprendemos a **estructurar visualmente una página**.

---

## ¿Qué es la maquetación web?

La **maquetación web** consiste en **organizar los distintos elementos de una página web dentro del espacio disponible en la pantalla**.

!!! note "Definición"
    La maquetación web es el proceso de **colocar y distribuir los elementos de una página web** para construir su estructura visual.

La maquetación responde a preguntas como:

- ¿Qué elementos van arriba y cuáles abajo?
- ¿Qué elementos se colocan uno al lado del otro?
- ¿Cómo se reparte el espacio en la pantalla?
- ¿Cómo se adapta la página a distintos tamaños de pantalla?

---

## Maquetación y diseño: no son lo mismo

Es importante no confundir **maquetación** con **diseño visual**.

- **Diseño**:

    - colores
    - tipografías
    - estilos visuales
    - apariencia

- **Maquetación**:

    - colocación de los elementos
    - estructura de la página
    - distribución del espacio

!!! tip "Importante"
    La maquetación define **dónde va cada cosa**, no cómo se ve.

---

## Ejemplo sencillo de estructura (sin estilos)

Una página web típica puede tener esta estructura:

- Cabecera
- Menú de navegación
- Contenido principal
- Pie de página

```text
-------------------------
|       CABECERA        |
-------------------------
|        MENÚ           |
-------------------------
|      CONTENIDO        |
-------------------------
|     PIE DE PÁGINA     |
-------------------------
```

---

## El concepto de layout

En maquetación web se utiliza mucho el término **layout**.

!!! note "Layout"
    Un **layout** es la forma en la que se organizan y distribuyen los elementos de una página web en la pantalla.

El layout define:

- filas
- columnas
- zonas
- orden de los elementos

---

## Tipos de layouts más habituales

- **Layout de una columna**
- **Layout de varias columnas**
- **Layout flexible**

---

## Herramientas CSS para la maquetación

A lo largo de esta unidad aprenderemos distintas herramientas de CSS que permiten realizar la maquetación:

- El flujo normal de los elementos
- La propiedad `display`
- El modelo de caja (box model)
- El centrado de elementos
- El posicionamiento
- Flexbox (introducción)
