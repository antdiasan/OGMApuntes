# El centrado de elementos en CSS

Uno de los problemas más habituales al maquetar una página web es **centrar elementos**.  
En CSS no existe una única forma de centrar, ya que el método de centrado depende del tipo de elemento y del tipo de centrado que se quiera realizar.

Por este motivo, es importante entender **qué se está centrando** y **dentro de qué contenedor**.

---

## Centrado horizontal

### Centrado horizontal de texto y elementos en línea

Para centrar texto o elementos en línea dentro de un contenedor se utiliza la propiedad `text-align`.

```css
.contenedor {
  text-align: center;
}
```

Este método solo afecta al contenido **en línea** o al texto, no a los elementos en bloque.

---

### Centrado horizontal de elementos en bloque

Para centrar horizontalmente un elemento en bloque es necesario que tenga un ancho definido.

```css
.caja {
  width: 200px;
  margin: 0 auto;
}
```

Al usar `margin: auto`, el navegador reparte el espacio sobrante a ambos lados del elemento.

---

## Centrado vertical

### El problema del centrado vertical

El centrado vertical ha sido tradicionalmente uno de los aspectos más complicados en CSS, especialmente antes de la aparición de técnicas modernas.

No existe una solución única válida para todos los casos usando únicamente CSS clásico.

---

### Centrado vertical de texto en una sola línea

Una forma sencilla de centrar verticalmente texto (solo en una línea) es utilizar la propiedad `line-height`.

```css
.caja {
  height: 50px;
  line-height: 50px;
}
```

Este método solo funciona correctamente cuando el texto ocupa **una sola línea**.

---

### ¿Se puede usar padding para centrar verticalmente?

En algunos ejemplos se utiliza la propiedad `padding` para dar **la sensación** de que el contenido está centrado verticalmente.

```css
.caja {
  padding: 50px;
  text-align: center;
}
```

---

## Centrado usando position (contenido avanzado)

Otra técnica para centrar elementos consiste en utilizar la propiedad `position`.

```css
.caja {
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
}
```

Este método permite centrar un elemento tanto horizontal como verticalmente, pero es menos intuitivo y más complejo.

Actualmente se considera una técnica avanzada o de uso puntual.

---

> ℹ️ Nota (contenido avanzado)
>
> Antes de la aparición de Flexbox, una técnica utilizada para el centrado vertical
> consistía en simular el comportamiento de las tablas usando `display: table`
> y `display: table-cell` junto con `vertical-align: middle`.
>
> Actualmente este método se considera una solución antigua y no se recomienda
> para nuevos diseños.
