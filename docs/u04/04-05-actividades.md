# Actividades: el centrado de elementos en CSS

En estas actividades se trabajarán distintas situaciones habituales en las que es necesario **centrar elementos en una página web**, utilizando únicamente técnicas vistas hasta ahora, sin emplear Flexbox.

El objetivo es comprender **qué se puede centrar**, **cómo** y **por qué a veces no funciona**.

---

## Actividad 1: centrado de texto

Crea un archivo HTML con un `<div>` que contenga texto.

Aplica los estilos necesarios para que el texto aparezca centrado horizontalmente dentro del contenedor.

Responde:

- ¿Qué propiedad CSS has utilizado?
- ¿Se ha centrado el texto o el elemento?

---

## Actividad 2: centrado horizontal de una caja

Crea un `<div>` con una clase `caja` y aplícale un ancho fijo.

Centra la caja horizontalmente dentro de la página utilizando CSS.

Responde:

- ¿Qué condiciones son necesarias para que el centrado funcione?
- ¿Qué ocurre si no se define el ancho?

---

## Actividad 3: centrado con inline-block

Crea tres cajas y colócalas en horizontal usando `display: inline-block`.

Centra el conjunto de cajas dentro de su contenedor.

Responde:

- ¿Dónde se aplica el centrado?
- ¿Qué elemento es el que realmente se centra?

---

## Actividad 4: intentos de centrado vertical

Intenta centrar verticalmente una caja dentro de un contenedor utilizando únicamente propiedades como:

- `margin`
- `text-align`

Observa el resultado y responde:

- ¿Se consigue el centrado vertical?
- ¿Por qué crees que no funciona?

---

## Actividad 5: reflexión final

Explica con tus propias palabras:

- Por qué el centrado vertical es más complicado en CSS tradicional.
- Qué limitaciones tienen las técnicas vistas hasta ahora.
- Por qué crees que existen herramientas modernas para resolver este problema.
