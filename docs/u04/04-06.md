# Posicionamiento básico en CSS

Además del flujo normal del documento y de la propiedad `display`, CSS ofrece otra herramienta para controlar la colocación de los elementos en una página web: el **posicionamiento**.

El posicionamiento permite **mover elementos respecto a su posición original**, al contenedor o a la ventana del navegador.

---

## La propiedad position

La propiedad `position` indica **cómo se posiciona un elemento** y cómo se relaciona con el resto del layout.

Los valores más importantes que vamos a estudiar son:

- `static`
- `relative`
- `absolute`
- `fixed`

Cada uno de ellos modifica el comportamiento del elemento de una forma distinta.

---

## position: static

Es el valor por defecto de todos los elementos HTML.

Un elemento con `position: static`:

- sigue el flujo normal del documento
- no se ve afectado por las propiedades `top`, `left`, `right` o `bottom`
- no se puede mover manualmente

```css
.caja {
  position: static;
}
```

En la práctica, no suele declararse explícitamente porque es el comportamiento normal.

---

## position: relative

Un elemento con `position: relative` **mantiene su espacio en el flujo**, pero puede desplazarse respecto a su posición original.

```css
.caja {
  position: relative;
  top: 20px;
  left: 30px;
}
```

El desplazamiento se realiza:

  - respecto a la posición original del propio elemento
  - sin afectar al resto de elementos

Aunque el elemento se mueva visualmente, **el espacio original sigue reservado**.

---

## position: absolute

Un elemento con `position: absolute` **sale del flujo normal del documento**.

```css
.caja {
  position: absolute;
  top: 20px;
  left: 30px;
}
```

El elemento se posiciona respecto a:

  - el contenedor más cercano con `position` distinta de `static`
  - o la ventana del navegador si no existe ese contenedor

Al salir del flujo:

  - no reserva espacio
  - puede superponerse a otros elementos

---

## position: fixed

Un elemento con `position: fixed` se posiciona respecto a la **ventana del navegador**.

```css
.caja {
  position: fixed;
  bottom: 20px;
  right: 20px;
}
```

Este tipo de posicionamiento se utiliza, por ejemplo, para:

  - menús fijos
  - botones flotantes
  - cabeceras que permanecen visibles al hacer scroll

El elemento no se mueve al desplazarse la página.

---

## Propiedades de desplazamiento

Las propiedades más habituales para desplazar elementos posicionados son:

- `top`
- `left`
- `right`
- `bottom`

Estas propiedades **solo funcionan** si el elemento tiene un valor de `position` distinto de `static`.

---

## Uso del posicionamiento en la maquetación

El posicionamiento es una herramienta potente, pero debe usarse con cuidado.

Se utiliza principalmente para:

  - pequeños ajustes de posición
  - elementos flotantes o superpuestos
  - componentes específicos de la interfaz

No es la mejor opción para construir layouts completos, tarea que se realiza hoy en día con Flexbox u otras técnicas modernas.
