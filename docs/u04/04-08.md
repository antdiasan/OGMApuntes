# Introducción a Flexbox para maquetación

Hasta ahora hemos visto distintas técnicas de maquetación en CSS basadas en el flujo normal del documento, `display`, `inline-block` y posicionamiento.
Estas técnicas funcionan, pero presentan limitaciones cuando se quieren crear layouts más flexibles y adaptables.

Para resolver estos problemas surge **Flexbox**, un modelo de maquetación pensado específicamente para **alinear y distribuir elementos de forma sencilla**.

---

## ¿Qué es Flexbox?

Flexbox (Flexible Box Layout) es un modelo de diseño de CSS que permite organizar elementos dentro de un contenedor de manera flexible.

Su objetivo principal es facilitar tareas como:

  - crear columnas o filas,
  - alinear elementos,
  - repartir el espacio disponible,
  - centrar contenido de forma sencilla.

A diferencia de técnicas anteriores, Flexbox está pensado desde el inicio para adaptarse a distintos tamaños de pantalla.

---

## El contenedor flex y los elementos flex

Para trabajar con Flexbox es necesario distinguir entre dos conceptos:

- **Contenedor flex**: es el elemento al que se le aplica `display: flex`.
- **Elementos flex**: son los elementos hijos directos del contenedor flex.

Al activar Flexbox en un contenedor, sus hijos cambian automáticamente su forma de comportarse y colocarse.

```css
.contenedor {
  display: flex;
}
```

---

## Ejes principales en Flexbox

Flexbox trabaja con dos ejes:

  - **Eje principal**: dirección en la que se colocan los elementos.
  - **Eje secundario**: eje perpendicular al principal.

Por defecto:

  - el eje principal es horizontal,
  - el eje secundario es vertical.

Comprender estos ejes es fundamental para entender cómo funcionan las propiedades de alineación.

---

## Dirección de los elementos: flex-direction

La propiedad `flex-direction` indica la dirección del eje principal.

Valores más habituales:

  - `row`: los elementos se colocan en fila (valor por defecto).
  - `column`: los elementos se colocan en columna.

```css
.contenedor {
  display: flex;
  flex-direction: row;
}
```

Cambiar esta propiedad modifica completamente la disposición del layout.

---

## Alineación en el eje principal: justify-content

La propiedad `justify-content` controla cómo se distribuyen los elementos a lo largo del eje principal.

Permite, entre otras cosas:

  - centrar elementos,
  - repartir espacio entre ellos,
  - colocarlos al inicio o al final.

```css
.contenedor {
  display: flex;
  justify-content: center;
}
```

---

## Alineación en el eje secundario: align-items

La propiedad `align-items` controla la alineación de los elementos en el eje secundario.

Es especialmente útil para:
  
  - centrar elementos verticalmente,
  - alinear elementos por arriba o por abajo.

```css
.contenedor {
  display: flex;
  align-items: center;
}
```

---

## El centrado completo con Flexbox

Una de las ventajas más conocidas de Flexbox es la facilidad para centrar elementos tanto en horizontal como en vertical.

```css
.contenedor {
  display: flex;
  justify-content: center;
  align-items: center;
}
```

Con solo estas propiedades, el contenido queda perfectamente centrado dentro del contenedor.

---

## Ventajas de Flexbox frente a técnicas anteriores

Flexbox permite resolver de forma sencilla problemas habituales de maquetación:

- elimina los espacios indeseados de `inline-block`,
- simplifica la creación de columnas,
- facilita el centrado vertical,
- mejora la adaptación a distintos tamaños de pantalla.

Por este motivo, Flexbox es una de las herramientas más utilizadas en la maquetación web actual.
